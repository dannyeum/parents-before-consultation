## ① Firebase 설정 (5분)

1. https://console.firebase.google.com → **프로젝트 만들기**
2. **Firestore Database** → **데이터베이스 만들기** → 테스트 모드
3. **프로젝트 설정(⚙️)** → **앱 추가(</> 웹)** → apiKey, authDomain, projectId 복사

---

## ② Google Gemini API 키 발급 (3분, 완전 무료!)

1. https://aistudio.google.com/app/apikey 접속 (구글 계정 로그인)
2. **Create API Key** 클릭 → 키 복사: `AIza...`
3. 결제 설정 없이 즉시 사용 가능!

| 모델 | 무료 한도 | 추천 |
|------|-----------|------|
| `gemini-1.5-flash` | 분당 15회 / 하루 1,500회 | ✅ 기본값 |
| `gemini-1.5-pro`   | 분당 2회 / 하루 50회 | 더 정밀 |

> 반 30명 AI 분석 = 30회 → 무료 한도 내 충분!

## ③ .env.local 파일 작성

```bash
# Google Gemini (무료 AI)
GOOGLE_GEMINI_API_KEY=AIza여기에_구글_키_입력

# Firebase
NEXT_PUBLIC_FIREBASE_API_KEY=AIza여기에입력
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your-project.firebaseapp.com
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your-project-id
```

## ⑤ Vercel 배포

1. https://vercel.com → GitHub 연결 → repo 선택
2. **Environment Variables** 탭 → 4개 입력:

| Key | Value |
|-----|-------|
| `GOOGLE_GEMINI_API_KEY` | `AIza...` (구글 Gemini 키) |
| `NEXT_PUBLIC_FIREBASE_API_KEY` | `AIza...` (Firebase 키) |
| `NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN` | `xxx.firebaseapp.com` |
| `NEXT_PUBLIC_FIREBASE_PROJECT_ID` | `your-project-id` |

3. **Deploy** → 1분 후 완료! `https://ptc-app.vercel.app`

## ❓ FAQ

**Q: 한도 초과 시?**
앱에서 "잠시 후 다시 시도해주세요" 메시지 표시. 수분 후 재시도.

**Q: 더 많이 쓰고 싶다면?**
Google AI Studio에서 결제 추가. Gemini Flash는 100만 토큰당 약 $0.075로 매우 저렴.

**Q: 데이터 영구 저장?**
Firebase Firestore에 영구 저장. 1차·2차 모두 유지.
